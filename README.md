# Roboter
This is Robotics school project developed by a friend of mine and myself. The Java-Project in this repository is the software to control the 3d-printed gripper with Dynamixel servo. The project is using source code provided by Dynamixel (ROBOTIS Dynamixel SDK (Protocol1.0/2.0)) under the Apache License Version 2.0.

The source codes in the dynamixel package is basically the code provided by Dynamixel. It might be possible that minore changes have been made to those. The original version can be found her: https://github.com/ROBOTIS-GIT/DynamixelSDK

We are using the dynamixel AX-12A servo and the USB2AX v3.2b controller.
